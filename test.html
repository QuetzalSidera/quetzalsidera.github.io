API文档

我们的API基于RESTful设计，使用JSON格式传输数据，支持实时对话和批量处理。

基础对话接口

POST /api/v1/chat/completions
Content-Type: application/json
Authorization: Bearer YOUR_API_KEY

{
  "message": "你好，请介绍一下你自己",
  "conversation_id": "optional_id",
  "max_tokens": 1000
}

流式响应接口

GET /api/v1/chat/stream
Authorization: Bearer YOUR_API_KEY

使用指南

快速开始使用我们的AI对话平台：

  1. 注册开发者账号并获取API密钥
  2. 安装对应的SDK或直接调用REST API
  3. 集成到您的应用程序中
  4. 测试并优化对话体验
